# Background Remove

A simple Python script for automatic background removal from images.

## Description

This project uses the `rembg` library to remove backgrounds from PNG/JPG images using artificial intelligence.

## Installation

Install the required dependencies:

```bash
pip install rembg pillow
```

## Usage

1. Place the source image in the project directory named `image.png`
2. Run the script:

```bash
python background_remove.py
```

The resulting image will be saved as `output.png` with the background removed.

## Dependencies

- `rembg` - background removal library
- `Pillow` - image processing
